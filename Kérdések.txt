KERESŐ STRATÉGIA
1. találjunk egy legalább kétszavas kifejezést
2. jelenítsük meg az összes címet, amiben benne van, szűkítsük a keresést, amíg meg nem lesz egy pontos definíció vagy tétel vagy valami, és akkor
3. a vele kapcsolatos összes tudnivalót jelenítsük meg

BÖNGÉSZŐ STRATÉGIA
1. Jelenítsük meg az összes fejezetet
2. Jelenítsük meg annak a tartalmát, amire kíváncsiak vagyunk
3. Megkönyvjelzőzhessük, ahol tartunk.


1. Keresés a definíciók = fogalmak között => RÁKATT => 2. Részletezés megjelenítése a kérdésekkel tagoltan
2. Keresés a fejezetek tartalmában (kulcsszavakkal együtt) => RÁKATT a fejezet megjelenítése az adott helyen


+ háromszög magasságvonala => DEFINÍCIÓ, ELJÁRÁS (szerkesztése)
+ henger térfogata => DEFINÍCIÓ(?), KÉPLETEK(!)
+ két szám LNKO-ja => DEFINÍCIÓ, ELJÁRÁS (Euklideszi algoritmus), JELÖLÉS
+ Pitagorasz tétele => TÉTEL, KÉPLET
+ függvény deriváltja => DEFINÍCIÓ, KÉPLETEK(!), TÉTELEK
+ Skatulya-elv => TÉTEL,FELADATOK
+ binomiális tétel => TÉTEL, KÉPLET
+ Zorn axióma => AXIÓMA
+ V. posztulátum => AXIÓMA
+ pakolási probléma => FELADAT, MEGOLDÁS: ELJÁRÁS
+ ZFC aximómák
+ másodfokú egyenlet megoldóképlete



- Hogyan jelöljük ... ? => többféle lehetséges, akár historikusan is, JELÖLÉS
- Mit jelent? Hogyan definiáljuk ... ? => többféle lehetséges, DEFINÍCIÓ
- Hogyan számítható ki ... ? => a függvény értéke, a térbeli idom mértéke, algoritmus, KÉPLET, ELJÁRÁS
- Hogyan hozható létre (generálható, előállítható, szerkeszthető, ábrázolható) ... ? => ELJÁRÁS
- Hogyan oldható meg ... ? => feladat, egyenlet, egyenlőtlenség, probléma, -rendszer, algoritmus használata => ELJÁRÁS, MÓDSZER
- Tipikus példák feladatok a ...-vel kapcsolatban. => FELADAT
---
- Mit tudunk róla? => AXIÓMÁK, TÉTELEK
- Mit jelent a ... tétel, fogalom, jelölés ? => Magyarázat, szemléltetés, illusztráció
- Hogyan bizonyítható a ... tétel/állítás ? => BIZONYÍTÁS
- Mire jó? Hogyan alkalmazható/használható ... (akár egy tétel is) ? => keresés feladatok megoldásaiban, MÁS tételek bizonyításaiban, BIZONYÍTÁS, MEGOLDÁS
- Mi a szükséges / elégséges feltétele ...? => TÉTEL
